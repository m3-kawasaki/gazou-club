# 画像処理同好会 公式サイト

**公開ページ： https://m3-kawasaki.github.io/gazou-club/**

技術系出身のシニアが集う「画像処理同好会」の公式ウェブサイトです。
写真加工・画像処理・PC相談・生成AIをテーマに、偶数月の第1土曜14時からZoomで例会を開催しています。

## サイトの内容

| セクション | 内容 |
|---|---|
| ヒーロー | 粒子アニメーション（マウスに反応）付きのトップ画面 |
| 会について | 会の紹介と会員数などの統計カード |
| 例会のご案内 | **次回例会日を自動計算**して表示（偶数月・第1土曜・14時）。カウントダウンと年間スケジュール付き |
| 活動テーマ | 写真加工／PC・デジタル相談／生成AI研究の3本柱 |
| ビフォー・アフター | モノクロ写真のAIカラー化などを**左右スライダーで見比べ**られる比較コーナー |
| 作品ギャラリー | 会員の作品を展示。クリックで拡大表示（ライトボックス） |
| 参加者の声 | 会員の感想 |
| 入会のご案内 | 問い合わせ先（メールリンク） |

## ファイル構成

```
gazou-club/
├── index.html   … ページ本体（文章の修正はこのファイル）
├── style.css    … デザイン（色・レイアウト）
├── script.js    … 動き（例会日の計算・スライダー・アニメーション）
└── images/      … 掲載写真
```

## よくある更新のしかた

- **例会の日程** … 変更不要。`script.js` が「偶数月の第1土曜14時」を毎回自動計算します。開催ルール自体を変える場合は `script.js` 冒頭の `nextMeeting` 関数を修正してください。
- **ギャラリーの写真の差し替え** … 新しい画像を `images/` に追加し、`index.html` の「ギャラリー」セクションにある `<img src="images/～">` と説明文を書き換えます。
- **ビフォー/アフターの写真** … `index.html` の「ビフォー/アフター」セクションで、`compare-after`（処理後）と `compare-before` 内（処理前）の2枚を指定しています。
- **参加者の声・文章の修正** … `index.html` 内の該当箇所を直接編集します。

## 公開のしくみ

GitHub Pages を利用しています。`main` ブランチにファイルを push（またはGitHub上で編集・コミット）すると、1〜2分ほどで自動的に公開ページへ反映されます。

## 動作確認（ローカル）

フォルダ内で以下を実行し、ブラウザで `http://localhost:8000` を開くと公開前に確認できます。

```
python -m http.server 8000
```

## 配慮している点

- シニア向けに基本フォントを大きめ（18px）、コントラスト高めに設定
- スマートフォン・タブレット対応（レスポンシブ）
- OSの「視覚効果を減らす」設定（prefers-reduced-motion）に対応し、アニメーションを自動オフ
